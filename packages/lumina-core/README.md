# Lumina-Core

> Ritual engine + Dark Jester. Deterministic meets probabilistic.

Intelligence layer of the Aetherhaven Multiverse — dual-path engine governing how AetherMind agents think, decide, and surprise.

## Architecture

Deterministic Engine (Ritual Engine): Rule-based, predictable, stateful. Governs arc state transitions, agent sequencing, Hive ritual patterns.

Dark Jester Layer: Probabilistic, entropic, surprising. Injects controlled chaos. Feeds TransmissionFeed entropy. Cannot modify deterministic state directly.

## Integration

- Swift SDK — original iOS/macOS (sovereign local-first)
- TypeScript Bridge — Next.js 15 bindings for AetherMind
- Sandboxed: Dark Jester cannot write to persistent state
- All agent outputs schema-validated before display
- Prompt injection mitigations at all entry points
